# Portfolio Agent Rules

- Build one page section at a time. Do not redesign unrelated sections unless asked.
- Do not invent metrics, client names, awards, dates, testimonials, or project outcomes.
- Do not download or use external client logos, screenshots, stock imagery, or copyrighted assets. The user will provide approved assets.
- Keep all portfolio copy editable from a central data file.
- Use reusable components and keep the code clean, readable, and easy to extend.
- After each task, summarize changed files and mention anything still needed from the user.
- Run lint and build checks before marking a task complete.
